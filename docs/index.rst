betterspread
============

An async Python wrapper around `gspread <https://github.com/burnash/gspread>`_ that gives every cell and row
first-class async methods — read, write, clear, style, and delete without ever leaving your ``async``/``await`` flow.

.. code-block:: bash

   pip install betterspread
